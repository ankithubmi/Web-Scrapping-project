# Web Scraping Project

## Project Overview
This project uses Python and BeautifulSoup to scrape book data from a website and store it in a CSV file for analysis.

## Features
- Extract book information from webpages
- Parse HTML content
- Store data in CSV format
- Data ready for analysis

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas

## Project Structure
- web_scraping.py
- book.csv
- README.md

## How to Run
1. Install required libraries
   bash
   pip install requests beautifulsoup4 pandas
   

2. Run the script
   bash
   python web_scraping.py
   

## Dataset
- book.csv

## Output
- Scraped book data saved in CSV format
